# Pokemon ROM Editor

This is a rewrite of my original Pokemon ROM editor [link here](https://github.com/jakefordyce/PokemonROMEditor)

## Supported ROMS

Red (.gb)

Blue (.gb)

Gold (.gbc)

Silver (.gbc)

FireRed (.gba)

LeafGreen (.gba)
